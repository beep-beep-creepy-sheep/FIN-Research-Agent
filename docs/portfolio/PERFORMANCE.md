# Portfolio Performance

Performance analytics use only local prices and manual weights or quantities.

Supported outputs:

- Daily value series when enough position and price data exists.
- Cumulative return.
- Period return.
- Volatility.
- Maximum drawdown.
- Contribution by position where coverage is sufficient.
- Benchmark comparison as `missing_benchmark` unless aligned benchmark inputs exist.

The service does not create historical prices, forecast future returns, or fill missing prices with zero.
