# Portfolio Reports

Portfolio reports are deterministic summaries built from Stage 7 analytics and Stage 6 validation boundaries.

Sections:

- Portfolio summary.
- Holdings exposure.
- Risk snapshot.
- Performance summary.
- Alerts summary.
- Calendar summary.
- Data-quality limitations.

Reports preserve evidence and missing-data context. They do not provide rebalancing instructions, trading instructions, single-point valuation instructions, or guaranteed outcomes.

If local data is sparse, the report still renders with insufficient-data sections and explicit limitations.
